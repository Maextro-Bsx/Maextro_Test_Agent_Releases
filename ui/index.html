About Maextro Test Agent

Maextro Test Agent

📌 Overview

Maextro Test Agent is a desktop application that allows users to record and run tests quickly by using structured Excel templates.

🖥️ How to Use

1. Enter Username and Password

Provide your Maextro credentials:

  • Username
  • Password

⚠️ These credentials will be used to log in to the Maextro application during recording and execution.

2. Select Environment

Select the environment where you want to record or run the workflow:

  • DEV
  • TEST
  • SHS
  • Others (if available)

3. Template Selection

  • Templates are displayed based on the selected environment
  • Select an existing template for execution
  • ✅ Newly recorded templates will also appear in this list automatically

4. Download Template

Click "Download Template" to download the Excel file.

📂 The file can be saved locally anywhere on your system.

5. Fill or Modify Template

  • Open the downloaded Excel file
  • Enter or modify the required test data
  • Save the file after updating the values

6. Upload File

  • Upload the data-filled or modified Excel file
  • Ensure correct file selection before execution

7. Run Test

Click "Run Test" to start execution.

  • Workflow will run using uploaded data
  • Live execution logs will be displayed

8. View Latest Report

  • Once execution completes, click "View Latest Report"
  • A dialog/save prompt will appear
  • ✅ You can also save a local copy of the report if required

9. Record Workflow

The recording feature allows you to create new templates by capturing real user actions.

  • Enter Username, Password, and Environment
  • Click "Record" to begin recording
  • A browser will open and log in to the selected environment
  • Once the Maextro home page is visible, start performing actions manually
  • The recorder will capture:
    • Steps performed
    • Views/screens per step
    • Field inputs and data
    • Header data required for execution
  • After completing the workflow, close the browser
  • A confirmation popup will appear indicating the template is recorded
  • ✅ You can optionally save the template locally
  • ✅ The recorded template will automatically appear in the template dropdown for execution

⚠️ Important Notes

  • All fields marked with * are mandatory
  • Ensure correct environment selection before recording or running
  • Do not modify the Excel template structure
  • Only valid Excel files should be uploaded
  • Only one recording or execution can run at a time
  • Close the browser properly after recording to save the workflow
  • Recorded templates will be automatically available for execution
  • Reports can be viewed immediately or saved locally
