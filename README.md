# Maextro Automation Runner

## 📌 Overview

Maextro Automation Runner is a desktop application that allows users to execute test scenarios quickly by uploading structured Excel templates.

---

## 🖥️ How to Use

### 1. Enter Username and Password

Provide your Maextro credentials:

* Username
* Password

⚠️ These credentials will be used to log in to the Maextro application during execution.

---

### 2. Select Environment

Choose the environment where you want to run the workflow:

* DEV
* TEST
* SHS
* Others (if available)

---

### 3. Select Template

* Templates will be displayed in the dropdown based on the selected environment
* Choose the required template for your test execution

---

### 4. Download Template

Click **"Download Template"** to download the Excel file.

📂 The file will be saved in your system’s **Downloads folder**.

---

### 5. Fill the Template

* Open the downloaded Excel file
* Enter the required test data
* Save the file without changing its name

---

### 6. Upload File

* Upload the filled Excel file
* Ensure the filename matches the selected template

---

### 7. Run Test

Click **"Run Test"** to start execution.

You will see:

* Live execution logs
* Progress updates

---

### 8. View Report

Once execution is complete:

* Click **"View Report"**
* The report will open in your browser

---

## ⚠️ Important Notes

* All fields marked with * are mandatory
* Do not modify the structure of the template
* Ensure correct environment and template selection
* Only one execution can run at a time
